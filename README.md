# Undercover 🕵️

Jeu de déduction sociale inspiré du jeu Undercover.

## Règles
- 🏙️ **Citoyens** — partagent le même mot secret
- 🕵️ **Undercover** — a un mot similaire mais différent
- 👻 **Mr. White** — n'a aucun mot, doit bluffer

## Système de points
- Citoyen survivant : +1 pt
- Undercover gagnant : +2 pts
- Mr. White gagnant : +3 pts

## Fonctionnalités
- 16 mangas/animes, BD franco-belge, Marvel/DC, Cinéma
- 120+ paires cross-over entre univers
- Slider de probabilité cross-over
- Rôles cachés, Mr. White, ordre de passage aléatoire
- Classement inter-manches et podium

## Compatibilité
Fichier HTML standalone — compatible Safari iOS (ES5 pur).

Ouvrir `undercover.html` dans n'importe quel navigateur.
